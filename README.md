# pnpm-turborepo-project

## Getting started

```
$ npx pnpm i
```

## Commands

### App

#### Build / Generate

```
$ pnpm turbo run app#build
$ pnpm turbo run app#generate
```

#### Dev / Start

```
$ pnpm turbo run app#dev
$ pnpm turbo run app#start
```

### E2E

```
$ pnpm turbo run e2e#run
```

### Lint

Run lint script across all workspaces:

```
$ pnpm turbo run lint
```
